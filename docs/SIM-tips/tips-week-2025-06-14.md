# 🛠️ Tip of the Week:

##   Take Out the Trash.

If an LLM has given a wrong answer in a conversation, this wrong answer will pollute future answers. Because the entire conversation is sent back to the LLM everytime the wrong answer is becoming part of the context.

Sometimes asking for a correction does not have enough weight. Once you notice this pollution behaviour, start a new conversation and summarize your request in a clean prompt.
